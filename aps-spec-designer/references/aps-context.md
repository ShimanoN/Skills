## 電磁弁 動作真理表

| STATE | SOV-01 (DO-001) | EV-OUT (DO-002) | EV-BYP (DO-011) | 解説 |
|-------|----------------|-----------------|-----------------|------|
| 待機（0）| OFF（閉）| OFF（閉）| OFF（閉）| エアーなし |
| ①漏れ試験（1〜3）| ON（開）| **OFF（閉・封止）** | **OFF（閉）** | 全量が A-16 経由 |
| ①→②移行（4）| ON | OFF | **ON（先行全開！）** | **MFC 昇流前に必須** |
| ②流量検査（5〜7）| ON（開）| **ON（開・排気）** | ON（開）| MFC 60 L/min・FSM3 計測 |
| 結果表示（8,9）| OFF（閉）| OFF（閉）| OFF（閉）| エアー遮断・安全状態 |
| EMO（E）| **OFF（閉）** | OFF（閉）| OFF（閉）| 全遮断 |

> ⚠️ **重要：STATE_TRANSITION（4）** で EV-BYP を先行全開してから MFC 昇流すること。  
> 順序を逆にすると 60 L/min が流量計 A-16 を通過して **機器破損**の恐れあり。
